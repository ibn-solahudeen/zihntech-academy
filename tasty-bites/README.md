# 🍽️Tasty Bites - Restaurant Landing Page

Welcome to **Tasty Bites**, a modern and responsive restaurant landing page built as part of my Zinhtech Frontend Development Cohort.

This project showcases my understanding of HTML5 and CSS3 by creating a visually appealing and user-friendly restaurant website.

---

# 📖 About the Project

Tasty Bites is a responsive restaurant landing page designed to provide visitors with an attractive introduction to a restaurant. The page focuses on clean design, smooth navigation, and an engaging user experience.

---

# 🚀 Features

- Responsive design for desktop, tablet, and mobile devices
- Modern and clean user interface
- Hero section with call-to-action
- Navigation bar
- Food menu section
- About section
- Contact section
- Attractive layout using CSS
- Well-organized project structure
